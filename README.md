## About the Project

This project is part of the Open Source Software course.  
I selected Python as my open-source software and studied its origin, license, and how it works on Linux systems.

I also created 5 shell scripts to demonstrate basic Linux commands and scripting concepts.

## Scripts Description

1. script1.sh  
Displays system information like kernel version, user, uptime, and OS details.

2. script2.sh  
Checks whether Python is installed and shows its package details.

3. script3.sh  
Shows disk usage and permissions of important system directories.

4. script4.sh  
Analyzes a log file and counts how many times a keyword like "error" appears.

5. script5.sh  
Generates a simple open-source manifesto using user input.

## How to Run

chmod +x script1.sh  
./script1.sh  

chmod +x script2.sh  
./script2.sh  

chmod +x script3.sh  
./script3.sh  

chmod +x script4.sh  
./script4.sh /var/log/syslog error  

chmod +x script5.sh  
./script5.sh  

## Requirements

- Linux (Ubuntu / WSL)
- Basic terminal access
- Python installed
